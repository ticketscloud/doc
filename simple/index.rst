========================================
Простая интеграция для распространителей
========================================

Для того, чтобы продать :term:`билет` через TicketsCloud, нужно пройти следующие шаги.

1. Зарегистрировать кампанию на сайте → https://ticketscloud.org.
2. Получить :term:`API-ключ` от менеджеров TicketsCloud → :ref:`авторизация <basic/auth>`
3. Заключить :term:`сделку <сделка>` с :term:`организатором <организатор>` на :term:`мероприятие` → работа со сделками
4. Получить информацию о мероприятии → :ref:`работа с мероприятиями <simple/events>`
5. Создать и провести заказ → :ref:`работа с заказом <simple/orders>`
    a) Создание заказа
    b) Бронирование билетов
    c) Передача сервису информации о пользователе
    d) Указание платёжной системы
    e) Ожидание оплаты
    f) Подтверждение оплаты
